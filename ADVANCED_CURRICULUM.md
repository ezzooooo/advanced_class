# Advanced 커리큘럼 (심화반)

## 개요

| 항목     | 내용                                                                                             |
| -------- | ------------------------------------------------------------------------------------------------ |
| **주제** | 실무 역량 강화                                                                                   |
| **목적** | 실무에서 필수적인 테스팅, 배포, 모니터링, 성능 최적화 등 앱 출시 및 운영에 필요한 심화 기술 습득 |
| **목표** | 단순 앱 개발을 넘어 품질 관리, 자동화, 운영까지 가능한 실무형 개발자 역량 확보                   |
| **기간** | 7주                                                                                              |
| **일정** | 주차별 1시간 ~ 1시간 30분                                                                        |

---

## 주차별 커리큘럼

---

### 1주차: 테스팅 기초

> **목적**: 품질 높은 코드 작성을 위한 테스트 작성 능력 습득

#### 📚 이론 (30분)

| 주제            | 내용                                         |
| --------------- | -------------------------------------------- |
| 테스팅의 필요성 | 버그 조기 발견, 리팩토링 안정성, 협업 신뢰도 |
| 테스트 종류     | Unit Test vs Widget Test vs Integration Test |
| 테스트 피라미드 | 각 테스트의 비용과 효과                      |
| AAA 패턴        | Arrange-Act-Assert 구조                      |

#### 💻 실습 (30~45분)

| 실습             | 내용                             |
| ---------------- | -------------------------------- |
| Unit Test 작성   | 순수 함수, 비즈니스 로직 테스트  |
| Widget Test 작성 | 간단한 UI 컴포넌트 테스트        |
| Mocking 기초     | Mocktail을 활용한 의존성 Mocking |

#### 🎯 학습 목표

- [ ] `flutter test` 명령어 사용법 이해
- [ ] `test`, `expect`, `group` 함수 활용
- [ ] `WidgetTester`를 활용한 UI 테스트 작성
- [ ] Mock 객체 생성 및 활용

#### 📝 과제

- 본인 프로젝트의 유틸 함수 2개 이상 Unit Test 작성

### 2주차: 앱 성능 최적화

#### 📚 이론 (30분)

| 주제                | 내용                                         |
| ------------------- | -------------------------------------------- |
| Flutter 렌더링 원리 | Widget → Element → RenderObject              |

| const의 중요성      | const 생성자가 성능에 미치는 영향            |
| Key의 역할          | Widget 재사용과 Key의 관계                   |

#### 💻 실습 (30~45분)

| DevTools 활용  | Performance, Memory 탭 사용법            |
| 성능 문제 진단 | 실제 느린 화면 분석하기                  |
| 최적화 적용    | const, RepaintBoundary, ListView.builder |

#### 🎯 학습 목표

- [ ] Flutter DevTools 실행 및 활용
- [ ] 불필요한 rebuild 찾고 수정하기
- [ ] `RepaintBoundary` 적용 시점 이해
- [ ] `ListView.builder` vs `ListView` 차이 이해

#### 📝 과제

- 본인 프로젝트에서 성능 개선 포인트 1개 이상 찾아 수정

1. **테스트 작성**: 주요 비즈니스 로직에 대한 테스트 코드 작성 가능

5. **운영**: Remote Config를 활용한 동적 앱 제어
